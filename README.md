## 🌟 Features

- **Homepage** - Welcome section with statistics, quick links, and announcements
- **About Section** - College history, mission, vision, leadership, and infrastructure
- **Programs Section** - List of undergraduate, postgraduate, and diploma programs with filtering
- **Registration Section** - Multi-step application form with validation and success/failure notifications

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling, animations, and responsive design
- **JavaScript** - Form validation, interactivity, and dynamic content
- **Font Awesome** - Icons (CDN)

## 🚀 How to Run Locally

1. **Download or Clone** the repository
2. **Open** `index.html` in any web browser

No build tools or server required! This is a static website.

## 🌐 Deploying to GitHub Pages

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to **Settings** → **Pages**
4. Under "Source", select **main** branch
5. Click **Save**
6. Your site will be live at `https://yourusername.github.io/repository-name/`

## 📝 Form Validation Rules

- **Name**: Letters only (2-50 characters)
- **Email**: Valid email format
- **Phone**: 10-digit number
- **PIN Code**: 6-digit number
- **Age**: Between 16-30 years
- **File Size**: Maximum 2MB
